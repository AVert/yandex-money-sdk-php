Подробная инструкция по запуску примеров:
1) Установить WAMP-сервер. Мы пробовали на простом сервере The Uniform server, который даже устанавливать не нужно (http://en.wikipedia.org/wiki/The_Uniform_Server). Нужно скачать его, распаковать архив в какой-нибудь каталог и запустить Start.exe. Затем запустить Apache сервер, нажав на иконку The Uniform server в системном трее и включить сервер (Start Apache server).
2) Удалить все файлы из каталога www сервера, кроме .htaccess, и скопировать туда файлы примеров.
3) Скопировать в каталог www каталог yamoney c файлами ym.php и файл цепочки сертификатов ym.crt.
4) Получить идентификатор приложения по адресу https://sp-money.yandex.ru/myservices/new.xml, указав при регистрации следующие обязательные данные:
	- название: любое, но помните, что его будут видеть пользователи при подтверждении прав на доступ к их деньгам
	- адрес сайта: http://localhost/
	- Redirect URI: http://localhost/redirected.php
5) Прописать в файл consts.php следующие данные:
	- константа CLIENT_ID - идентификатор приложения
	- константа REDIRECT_URL - http://localhost/redirected.php
	- константа CERTIFICATE_CHAIN_PATH - абсолютный путь к файлу ym.crt. Например: D:\UniServer\www\yamoney\ym.crt 
5) Открыть в браузере страничку http://localhost/index.php и дальше следовать инструкциям.

Внимание: для гарантированного успешного запуска лучше не менять каталоги и имена файлов, чтобы не перепрописывать подключения файлов в примерах (require'ы).
